**🐰 ¡Bienvenido a Rabbit Form Validation Library!**

Esta es la documentación oficial de Rabbit Form Validation Library.
Aquí encontrarás todo lo necesario para implementar la validación declarativa de formularios usando nuestra librería, que incluye:

- 🐇 **Validación de formularios declarativa con datasets**
- 🚀 **Requiere un mínimo de JavaScript**
- 🔧 **Reglas de validación personalizables**
- 📄 **Integración clara con HTML**
- 🔗 **Fácil de extender para validaciones complejas**

## Primeros pasos

Para empezar a usar **Rabbit Form Validation Library**, tienes dos opciones:

1. **Descargar los archivos directamente desde el repositorio**:  
   Puedes clonar o descargar los archivos del repositorio en GitHub:
   [Descargar desde GitHub](https://github.com/tu_usuario/rabbit-form-validation-library)

2. **Instalar con npm**:  
   Si prefieres usar npm para gestionar tus dependencias, simplemente ejecuta el siguiente comando:

   ```bash
   npm install rabbit-validation-library
